To apply the rounded button effect simply add one of the following classes to your element:

blue_button, green_button, gray_button, red_button

By default there is no rounding, to add rounding simply add one of the following classes before the color class to the same element:

_round_5, _round_10, _round_15, _round_20, _round_25, _round_30

Example Usage: class="_round_5 blue_button"

The above will round the border by 5px and apply the style of the blue button
